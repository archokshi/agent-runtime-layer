# Code of Conduct

We want Agent Runtime Layer to be a respectful and useful project for developers working on agent observability and optimization.

## Expected Behavior

- Be kind and direct.
- Assume good intent.
- Focus criticism on ideas, code, and evidence.
- Help keep claims precise, especially around measured vs estimated performance.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing someone else's secrets, private traces, credentials, or proprietary code.
- Misrepresenting estimated results as measured results.

## Enforcement

Maintainers may edit, hide, or remove comments and may limit participation when behavior harms the project or community.
