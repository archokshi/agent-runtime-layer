Install Agentium

One command installs everything — Docker, Python, the SDK, and agent hooks — automatically.

macOS and Linux users

1
Open your terminal and cd into your project folder
2
Run the command below — Docker, Python and the SDK install automatically if missing
curl -sSL https://agent-runtime-layer.vercel.app/install.sh | bash
3
Your browser opens to localhost:4001 — dashboard is live, hooks are installed
4
Run your agent normally — traces appear in the dashboard automatically

Windows users

Option A — WSL (recommended): Open WSL terminal, paste the curl command above.

Option B — PowerShell: Install Docker Desktop first, then run in PowerShell:

curl.exe -O https://agent-runtime-layer.vercel.app/docker-compose.yml
docker compose up -d
Start-Process "http://localhost:4001"

Option C — Git Bash: Paste the curl command at the top of this page directly into Git Bash.
