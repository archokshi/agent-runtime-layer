Install Agentium

One command installs everything — Docker, Python, SDK, and hooks for Claude Code and Codex — automatically.

macOS / Linux / WSL

1
Open your terminal
2
Run the command below and press Enter
curl -sSL https://agent-runtime-layer.vercel.app/install.sh | bash
3
Dashboard opens at localhost:4001 — Claude Code and Codex hooks installed globally
4
Run your agent normally — every session appears in the dashboard automatically

Windows (PowerShell)

🐳 Requires Docker Desktop to be installed and running first. Everything else installs automatically.
1
Open PowerShell
2
Run the command below and press Enter
Set-ExecutionPolicy Bypass -Scope Process -Force; irm https://agent-runtime-layer.vercel.app/install.ps1 | iex
3
Dashboard opens at localhost:4001 — Claude Code and Codex hooks installed globally
4
Run your agent normally — every session appears in the dashboard automatically

Also works via WSL or Git Bash using the macOS/Linux command above.

What installs automatically

Both commands handle everything end to end:

  • Docker — checked and started (or installed if missing)
  • Agentium backend + dashboard — pulled from Docker Hub
  • Python — installed via Homebrew / apt / winget if missing
  • agentium-tracer SDK — installed via pip
  • Claude Code hooks — installed globally (captures all sessions)
  • Codex hooks — installed globally (captures all sessions)
  • Dashboard opens automatically at localhost:4001

Troubleshooting

Dashboard not opening?
Go to http://localhost:4001 in your browser. If blank, run: docker compose -f ~/.agentium/docker-compose.yml up -d
No traces appearing after running Claude Code or Codex?
Reinstall hooks globally by opening a new terminal and running:
python -m agent_runtime_layer.cli integrations install claude-code --global
python -m agent_runtime_layer.cli integrations install codex --global
Python not found after install?
Open a new terminal window — PATH updates only take effect in new sessions. Then run the installer again.
Docker Desktop not running on Windows?
Open Docker Desktop from your Start menu, wait for the whale 🐳 icon in the taskbar, then run the PowerShell command again.
Want to uninstall?
curl -sSL https://agent-runtime-layer.vercel.app/uninstall.sh | bash
