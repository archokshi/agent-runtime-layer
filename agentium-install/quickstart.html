Agentium

5-Minute Quickstart

See what your coding agent costs, where it stalls, and what to fix — in under 5 minutes. No code changes required.

1
Install Agentium
~1 min
macOS / Linux
Windows (WSL)
Windows (PowerShell)
curl -sSL https://agent-runtime-layer.vercel.app/install.sh | bash

Docker is installed automatically if you don't have it. The dashboard opens at localhost:4001 when ready.

curl -sSL https://agent-runtime-layer.vercel.app/install.sh | bash

Open your WSL terminal and paste the command above. Docker is installed automatically.

Install Docker Desktop first, then run in PowerShell:

curl.exe -O https://agent-runtime-layer.vercel.app/docker-compose.yml docker compose up -d Start-Process "http://localhost:4001"
Verify it's running: open http://localhost:4001 — you should see the Agentium dashboard.
2
See your first trace
~1 min

Go to localhost:4001/runs and click Import demo trace.

You'll immediately see a real agent session: token cost per call, tool wait time, repeated context percentage, and ranked recommendations — no agent connection needed yet.

3
Connect your agent
~2 min

Install the SDK:

pip install agent-runtime-layer

Install hooks in the repo where you run Claude Code:

agent-runtime integrations install claude-code --repo /path/to/your/repo

Run Claude Code normally — every prompt, tool call, and response is captured automatically. Open localhost:4001/runs and your run appears within seconds.

Using Codex instead?

agent-runtime integrations install codex --repo /path/to/your/repo
4
Enable optimizations
~1 min

Open localhost:4001/settings. Your plan is already set to Pro — all features are unlocked for the alpha period.

Toggle on Context Optimizer → Save → run your agent again. The Gains since enabled card on the overview page shows your before/after delta.

Typical result: −43% tokens · −43% cost/run · retries eliminated
