Why FastAPI + SQLite?
Zero external dependencies. Self-contained Docker image. Backs up to a single file. Runs on any machine. Enterprise can deploy on-prem with zero cloud footprint.
Hook mechanism (Phase 1.8 enforcement)
PreToolUse hook fires before every agent tool call. Budget Governor checks session cost + retry count. Returns blocked=true if limits exceeded. Claude Code stops the run instantly.
Context Memory proxy (Phase 1.9)
Intercepts POST /v1/messages. SHA-256 fingerprints stable blocks. Injects cache_control: ephemeral on matched blocks. Forwards to Anthropic. Records cache_read_input_tokens from response.
Settings-first Control Plane
SDK calls GET /api/settings on AgentRuntimeTracer init. Reads optimizer_enabled, budget limits, plan tier. Applies automatically — no code changes needed after the dashboard toggle is flipped.

Security & Privacy
Backup
Security model — self-hosted by design
Agents process proprietary code. None of it leaves the machine.
🔒
100% local storage
All trace data, prompts, tool outputs stored in local SQLite at ~/.agentium/data/. No external database, no cloud sync, no telemetry. You own 100% of the data.
🌐
Zero external connections
Backend makes no external API calls. Dashboard makes no external calls. The only outbound traffic is when the proxy forwards to Anthropic — and that only happens because you set the env var.
🔑
Automatic secret redaction
The SDK redacts obvious secrets (API keys, tokens, passwords matching common patterns) before persisting to SQLite. Redaction runs before storage — not as a post-processing step.
🐳
Docker isolation
All services run in isolated Docker containers. Only localhost:8000 and localhost:4001 are exposed — no external port binding by default. Enterprise can add reverse proxy + auth layer on top.
📋
SOC 2 roadmap
SOC 2 Type II preparation begins post-seed as part of the enterprise readiness phase. All current architecture decisions are made with SOC 2 controls in mind.
🗑
Complete removal
docker compose down removes all services. rm -rf ~/.agentium removes all data. No account to delete, no support ticket required.
